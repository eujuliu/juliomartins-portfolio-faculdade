Sobre mim

Sou um engenheiro de software autodidata com paixão por tecnologia e resolução de problemas. Desde 2019, venho desenvolvendo minhas habilidades por meio de recursos online gratuitos e prática diária. Tenho 1 ano de experiência profissional e 2 anos em projetos open source e pessoais. Sou altamente motivado e sempre em busca de novas oportunidades para aprender e aplicar minhas habilidades em ambientes desafiadores e dinâmicos.

Especialidades

Minha especialidade é o desenvolvimento de aplicativos JavaScript/TypeScript, com foco em:

  • Front-End: Criação de interfaces de usuário interativas com React.js e Vue.js.
  • Back-End: Desenvolvimento de APIs REST e GraphQL com Node.js e Nest.js.
  • Mobile: Desenvolvimento de aplicativos nativos para Android e iOS usando React Native.
  • Full-Stack: Criação de aplicativos da web com renderização do lado do servidor e sites estáticos usando Next.js.
  • Bancos de Dados: Trabalho com bancos de dados relacionais (PostgreSQL, MySQL) e NoSQL (MongoDB), utilizando SQL ou ORMs.
  • Infraestrutura: Experiência com AWS e IBM, além de práticas de testes automatizados.

Portfólio

API de Restaurante

Uma API de restaurante, onde é possível registrar um novo usuário, fazer um pedido e realizar pagamentos. Esta API foi desenvolvida com Node.js, TypeScript, Prisma (PostgreSQL) e os princípios de SOLID e DDD. Os testes unitários e de integração foram realizados com Jest e Docker.

Encurtador de URL

Site de Notas

Neste site, o usuário pode criar notas usando Markdown, e elas ficam salvas no LocalStorage do navegador. Dessa forma, se o navegador for reiniciado, as notas permanecem salvas. Este site foi desenvolvido com React.js, Redux, JavaScript e SCSS.
